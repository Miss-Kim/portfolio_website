GRO

So, What Exactly is GRO?

Many apps are available today for kids and children, but not many for toddlers. Many parents want to improve their children learning skills in a fun and exciting way. With this in mind, this mini-study project aimed to create a foundation where small children can learn the basic (colour, animals and sounds) to more advanced words (vehicles, shapes, numbers) in a fun and engaging way.
